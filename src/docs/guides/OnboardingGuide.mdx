# Guia de Onboarding

Bem-vindo ao sistema! Este guia irá ajudá-lo a começar.

## Primeiro Acesso

1. Faça login com suas credenciais
2. Complete seu perfil
3. Configure suas preferências
4. Explore o dashboard

## Funcionalidades Principais

### E-mails

1. Crie templates personalizados
2. Gerencie sua lista de contatos
3. Envie e-mails em lote
4. Acompanhe métricas

### Tarefas

1. Organize suas listas
2. Defina prioridades
3. Acompanhe progresso
4. Colabore com a equipe

## Dicas e Truques

### Atalhos de Teclado

- `Ctrl + N`: Nova tarefa
- `Ctrl + E`: Novo e-mail
- `Ctrl + F`: Busca rápida
- `Esc`: Fechar modal

### Personalização

- Tema claro/escuro
- Layout adaptativo
- Notificações configuráveis
- Filtros personalizados

## Suporte

- Documentação completa
- Tutoriais em vídeo
- Chat de suporte
- Base de conhecimento
